# BennyAndPenny.com WCAG Update

## New direction

The portfolio accessibility program now follows the shared WCAG 2.2 AA specification. Start with this portfolio site, then apply the same standard to the Adventures store in its separate repository.

## Work completed

- Skip link, landmarks, focus visibility, reduced motion, forced colors, and shared target-size support.
- Keyboard mobile navigation with focus handling and Escape close.
- Accessible Contact form validation, errors, and status messages.
- External link labels and new-tab disclosure.
- Accessibility statement and support contact.
- Native footer preferences controller for text size, high contrast, paused motion, and readable font. Preferences are saved locally.
- Smoke test, browser axe audit framework, manual checklist, content author guide, and WCAG program documentation.
