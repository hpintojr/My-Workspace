## Free-First Hosting Direction

Use the lowest-cost stack possible at launch:

- Vercel Hobby for the Next.js site, account area, API routes, and Payload integration.
- Neon free Postgres for Payload data.
- Cloudflare R2 for private PDF, EPUB, and audiobook files.
- Cloudflare DNS for domain/DNS.
- GitHub for source control and deployments.
- Stripe with no monthly fee for checkout/payments.
- Mailjet free tier once account is unblocked.

Avoid paying for Payload Cloud, VPS hosting, or separate backend infrastructure unless usage forces it later.
