## R2 / Fulfillment Direction

Private Cloudflare R2 object key direction:

```txt
ebooks/book-1/home-infusion-day.pdf
ebooks/book-1/home-infusion-day.epub
audiobooks/book-1/home-infusion-day.mp3
```

Delivery model:

```txt
Stripe purchase completed
→ app creates order/access records in Payload
→ customer requests file
→ app checks purchase, expiration, and download count
→ app generates short-lived signed R2 URL
→ app increments download/access count
```

Initial download/access rule: max 3 downloads/access attempts per purchased file/link.
