# Manager-Only New Hire Activation Checklist

## Before issuing this kit

- [ ] Confirm the final company name and product brand.
- [ ] Confirm the current package names, pricing, and what is included in each package.
- [ ] Confirm all compensation and clawback language has been approved by counsel / leadership.
- [ ] Confirm approved scripts, email templates, text templates, and recording disclosures.
- [ ] Confirm the Sales HQ sub-account is ready.
- [ ] Confirm agent role uses **Only Assigned Data** and no unnecessary permissions.
- [ ] Confirm agent cannot access client sub-accounts, agency settings, payments, workflows, exports, or other agents’ leads.

## Create the new agent

- [ ] Create User account in Sales HQ.
- [ ] Assign Sales Agent role.
- [ ] Enable Only Assigned Data.
- [ ] Grant Sales Pipeline access only.
- [ ] Assign calendar and phone / email tools as appropriate.
- [ ] Add agent to manager notifications and coaching calendar.
- [ ] Send welcome email with Start Here document.

## Documents and acknowledgement

- [ ] Sales Partner Agreement complete.
- [ ] NDA / confidentiality / IP agreement complete.
- [ ] W-9 and payout information collected through secure system.
- [ ] Compensation policy acknowledged.
- [ ] CRM and data-security policy acknowledged.
- [ ] Compliance policy acknowledged.

## Training and certification

- [ ] Product training complete.
- [ ] Script roleplay passed.
- [ ] CRM practical passed.
- [ ] Demo observation complete.
- [ ] Certification scorecard signed.
- [ ] Live outreach approved.

## First 30 days

- [ ] Schedule daily check-ins for first three business days.
- [ ] Schedule weekly coaching.
- [ ] Audit first 10 CRM records.
- [ ] Audit first 10 outreach conversations.
- [ ] Review 30-day performance and continued access.
