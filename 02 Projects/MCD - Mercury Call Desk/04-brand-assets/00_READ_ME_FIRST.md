# Mercury Call Desk Brand Assets

## Current status

- The logo direction is cataloged and saved in this folder.
- A clean SVG reconstruction is included so the emblem and wordmark can scale cleanly.
- The SVG wordmark uses editable `<text>` elements. It is intentionally not outlined yet.

## Use these files

- `03-vector-svg/mcd-logo-horizontal-dark.svg` — white wordmark for dark backgrounds.
- `03-vector-svg/mcd-logo-horizontal-light.svg` — dark-navy wordmark for white/light backgrounds.
- `03-vector-svg/mcd-logo-icon.svg` — icon-only mark.
- `03-vector-svg/mcd-wordmark-editable.svg` — editable text-only wordmark.
- `03-vector-svg/mcd-logo-system-editable.svg` — full editable SVG system.

## Important

The SVG rebuild is a clean reconstruction based on the approved reference, not a mathematical trace of its raster pixels. Before final trademark or mass-production use, Hamilton should approve the exact icon geometry and lock the font. After that, export a second outlined-text release for systems that cannot preserve live text.
