# Mercury Call Desk Logo Asset Catalog

## Approved visual direction

- Winged circular Mercury-inspired emblem
- Continuous call / ECG waveform through the center
- Blue-to-cyan gradient icon treatment
- `MERCURY` set in a heavy uppercase geometric sans-serif treatment
- `CALL DESK` set in a lighter uppercase sans-serif treatment

## SVG production system

### `03-vector-svg`

- `mcd-logo-horizontal-dark.svg` — transparent SVG with white wordmark for dark backgrounds.
- `mcd-logo-horizontal-light.svg` — transparent SVG with navy wordmark for light backgrounds.
- `mcd-logo-icon.svg` — transparent vector icon-only mark.
- `mcd-wordmark-editable.svg` — editable SVG text wordmark.

The live-text versions remain the source masters for color and type changes.

## Provisional color system

## Editable text rules

- Keep `MERCURY` heavy and `CALL DESK` lighter.
- The current source uses `Montserrat` as a portable visual stand-in with Arial/Helvetica fallbacks.
- Confirm and license the final wordmark font before outlining a production version.
- Do not outline the editable master. Create a separately named outlined release after final font approval.
