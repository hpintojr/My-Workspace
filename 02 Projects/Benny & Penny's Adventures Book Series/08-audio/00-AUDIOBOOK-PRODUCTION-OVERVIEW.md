# Audiobook Production Overview

## Source rule
Create the audiobook script from the locked manuscript—not screenshots or draft Canva text.

## Pipeline
1. Lock manuscript/page order.
2. Create narration script with private page cues.
3. Create pronunciation and performance glossary.
4. Record or synthesize approved narration.
5. Edit master and export website MP3.
6. Perform script/audio QA.
7. Upload through the book’s website manifest.

## Deliverables
- narration script (Markdown)
- pronunciation/performance glossary
- cue sheet
- protected archival master
- finished website MP3
- QA and upload record
