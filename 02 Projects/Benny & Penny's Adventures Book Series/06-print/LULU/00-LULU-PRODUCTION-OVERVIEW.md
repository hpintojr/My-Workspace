# Lulu Production Overview — Paperback + Hardcover

## Final outputs per book/language
- Paperback interior PDF
- Paperback wrap-cover PDF
- Hardcover interior PDF
- Hardcover wrap-cover PDF
- Physical proof approval record

## Start final print work after
- manuscript is locked
- final page count is confirmed
- binding/paper/ink choices are settled
- the current Lulu template/specification has been recorded

## Never reuse
- paperback covers for hardcover
- an old cover template after page count changes
- customer eBook PDF as print interior
- Lulu production PDFs for another printer
