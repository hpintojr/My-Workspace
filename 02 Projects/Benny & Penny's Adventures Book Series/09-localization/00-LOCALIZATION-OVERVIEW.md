# Localization Overview

## Languages
- Spanish (`es`) — choose a release locale such as `es-US`, `es-MX`, or `es-ES`
- French (`fr`) — choose `fr-FR` or `fr-CA`
- German (`de-DE`)
- Italian (`it-IT`)
- Japanese (`ja-JP`)
- Russian (`ru-RU`)
- Simplified Chinese (`zh-Hans`)
- Traditional Chinese (`zh-Hant`)

## Workflow
Final English text → translator → native children’s editor → author approval → localized Canva layout → PDF/EPUB/audio script → language QA.

Translate each edition directly from frozen English source, never from another translation.

## Audiobook localization
Each language folder includes a page-by-page audiobook narration manuscript setup, pronunciation guide, production record, and language-specific website-audio manifest. Do not start recording until the locale translation is approved by a native-language children’s editor.
