## This kit covers
- Character, environment, wardrobe, and medical-prop locking
- English manuscript and page-by-page scene briefs
- Canva English master plus each localized layout
- Lulu paperback and hardcover production records
- Direct PDF, fixed-layout EPUB/KPF, and audiobook deliverables
- Narration scripts, pronunciation records, cue sheets, and audio QA
- Website upload manifests for the existing `pdf_path`, `epub_path`, and `audio_path` fields

## Workflow
`Source manuscript → page scene briefs → visual locks → Canva English layout → PDF/EPUB/audio script → locale layouts → Lulu production → QA/proofs → website uploads`

## Lulu control point
The kit contains complete paperback and hardcover preflight files. Before producing a final print cover or interior, record the current requirements/template for that exact Lulu product configuration in `06-print/LULU/01-LULU-LIVE-SPEC-CAPTURE.md`. Final cover dimensions depend on the final product choices and page count.

## Security / Git rule
Commit Markdown files, project manifests, approved text, and low-resolution review assets. Keep master art, family source photos, paid customer files, printer PDFs, and master audio in private storage or a private repository.
