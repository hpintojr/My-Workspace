# Canva Master Setup

## Folder structure
- `BPA 00 Series Control`
- `BPA 01 Locked Characters`
- `BPA 02 Locked Environments`
- `BPA 03 Books`
- `BPA 04 Lulu Print`
- `BPA 05 Digital Editions`
- `BPA 06 Localization`
- `BPA 07 Released Exports`

## Required copies per book
1. English visual/layout master
2. Direct website PDF copy
3. Lulu paperback interior copy
4. Lulu hardcover interior copy
5. One localized copy per language
6. Separate paperback and hardcover cover designs
7. Audiobook thumbnail/cover source

## Rules
- Build individual pages, not print-source spreads.
- Protect text zones before approving artwork.
- Never overwrite English master after a locale version is created.
- Export each print file only from its matching provider/binding Canva design.
